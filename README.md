# Inference Detection

A unified Inference Comparison system focused on MediaPipe and ControlNet open-pose landmark models, with RetinaFace face detection.

## Features

- **Face** — face landmark detection via MediaPipe Face Landmarker
- **Hands** — hand landmark detection via MediaPipe
- **Pose** — pose landmark detection via MediaPipe
- **RetinaFace** — face detection and landmarks via RetinaFace
- **YOLOv8 Pose** — pose estimation with sitting/standing classification
- **LiftOptimize** — squat & deadlift form checker with LSTM activity classification (standing, walking, lifting, bicep curl, pushup, pullup)
- Supports images, video files, and webcam input
- Results saved to `results/` as `{filename}_{detector}.{ext}`

## Setup

1. **Create and activate a virtual environment with python <=3.11.x**

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

## Usage

### Inference Comparison (`main.py`)

```bash
# ControlNet OpenPose detection
python main.py --model cn_op --save  --source "samples/band.mp4"

# RetinaFace detection
python main.py --model rf --save  --source "samples/band.mp4"

# MediaPipe
python main.py --model mp --view  --source 0

# YOLOv8 Pose (sitting/standing)
python main.py --model yv8pose --view  --source 0
```

### LiftOptimize — Squat & Deadlift Checker (`main_lstm.py`)

```bash
# Webcam with MediaPipe backend
python main_lstm.py --source 0 --runner mp

# Video file with YOLOv8 backend, save output
python main_lstm.py --source "samples/squat.mp4" --runner yv8 --save

# Custom YOLOv8 weights
python main_lstm.py --source 0 --runner yv8 --weights weights/yolov8s-pose.pt
```
---
### LSTM Training (`model_training/lstm.py`)

```bash
# Train from videos in training_data/{standing,walking,lifting,bicep,pushup,pullup}/
python model_training/lstm.py
```

Training settings like sequence length, batch size, learning rate, epochs, and pose confidence values are in `model_training/config.py`.

### Data Collection (`model_training/collect_data.py`)

```bash
# Extract landmark sequences from labelled videos and save .npy files
python model_training/collect_data.py
```

## Parameters

### `main.py`
```
 Parameter      Default     Description 

 `--source`     `Required`  Image/video path or webcam index (e.g. `0`) 
 `--model`      `Required`  choices=["mp", "rf", "cn_op", "yv8pose"]
 `--save`       `None`      Save in "results"
 `--view`       `None`      View while processing
 `--output_dir` `results`   Output folder
```

### `main_lstm.py`
```
 Parameter    Default                    Description

 `--source`   `0`                        Video path or webcam index
 `--runner`   `mp`                       Pose backend: mp or yv8
 `--weights`  `weights/yolov8s-pose.pt`  YOLOv8 weights (only with --runner yv8)
 `--save`     `None`                     Save output to results/
```

## Weights

MediaPipe `.task` model files must be stored in `weights/`. Download them via:
```
https://drive.google.com/drive/folders/1j1qoZIK7e7ecVOeXEGK5yrAQcIDEilf5?usp=sharing
```

YOLOv8 pose weights (`yolov8s-pose.pt` or `yolov8m-pose.pt`) are downloaded automatically by Ultralytics on first run.

The LSTM classifier (`activity_lstm.pt`) is generated by running `model_training/lstm.py`. Without it, LiftOptimize falls back to a rule-based activity detector.

## Output

Results are saved in the `results/` directory as `{filename}_{model}_{time_string}.{ext}`.

Trained LSTM weights are saved to `weights/activity_lstm.pt` after running `model_training/lstm.py`.
